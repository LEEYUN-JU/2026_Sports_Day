## 🏃‍♂️ 운동회 웹페이지 프로젝트

운동회 진행 시 참가자들이 모바일을 통해 쉽고 빠르게 정보에 접근할 수 있도록 웹페이지를 개발했습니다.
실시간 정보 확인과 간편한 입력 기능을 통해 행사 운영 효율성을 높이는 것을 목표로 했습니다.

<br/>

### 🧩 주요 기능

**🏠 메인 페이지**
- 전체 운동회 정보 요약 제공
- 주요 메뉴로 빠르게 이동 가능

**🔴 홍팀 / ⚪ 백팀 관리**
- 각 팀별 참가자 이름 입력 기능
- 종목별 참가자 구분 가능

**📝 점수표**
- 팀별 점수 실시간 확인
- 경기 결과 반영 가능

**📅 스케줄표**
- 시간대별 경기 일정 확인
- 참가자 및 운영진 모두 활용 가능

**⚙️ 관리자 시스템**
- 관리자 전용 페이지 별도 구성
- 관리자 로그인 시: 참가자 정보 수정 / 점수 및 일정 업데이트 가능
- 일반 사용자는 조회 기능만 제공

<br/>

### 🌐 서버 구성 & 성능

| 항목 | 내용 |
|------|------|
| 서버 | Firebase Realtime Database |
| 동시 접속 | 최대 30명 안정 동작 확인 |
| 환경 | 모바일 웹 중심 설계 |
| 접근 방식 | 별도 앱 설치 없이 접근 가능 |

<br/>

### 💡 기대 효과

- 📄 종이 없이 디지털로 운동회 운영 가능
- 😊 참가자 편의성 향상
- ⚡ 운영진 관리 효율 증가

## 🏃‍♂️ 運動会 Web プロジェクト

<br/>

### 🧩 主な機能

**🏠 メインページ**
- 運動会全体情報の一覧表示
- 各メニューへのクイックナビゲーション

**🔴 紅チーム / ⚪ 白チーム管理**
- チームごとの参加者名入力
- 種目別の参加者振り分け

**📝 スコアボード**
- チーム別リアルタイムスコア確認
- 試合結果の即時反映

**📅 スケジュール**
- 時間帯別の試合スケジュール
- 参加者・運営スタッフ共通で利用可能

**⚙️ 管理者システム**
- 管理者専用ページを別途用意
- ログイン後：参加者情報・スコア・スケジュールの編集が可能

## 🏃‍♂️ Sports Day Web Project

A mobile-first web app built for smooth sports festival management.
Real-time scores, team rosters, schedules, and admin controls — all in one place, no app install needed.

<br/>

### 🧩 Key Features

**🏠 Home Page**
- Overview of all event info at a glance
- Quick navigation to key sections

**🔴 Red Team / ⚪ White Team Management**
- Participant name entry per team
- Sorting participants by event

**📝 Scoreboard**
- Real-time score tracking by team
- Instant game result updates

**📅 Schedule**
- Time-based match schedule view
- Useful for both participants and staff

**⚙️ Admin System**
- Dedicated admin-only page
- Edit participants, scores, and schedules after login
- Regular users: view-only access

<br/>

### 🌐 Tech Stack & Performance

| Item | Detail |
|------|--------|
| Backend | Firebase Realtime Database |
| Capacity | Stable under 30+ simultaneous connections |
| Environment | Mobile-optimised web |
| Access | No app installation required |

<br/>

### 💡 Impact

- 📄 Fully paperless event management
- 😊 Improved experience for all participants
- ⚡ Reduced workload for event staff
